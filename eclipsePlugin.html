Telosys Eclipse Plugin

The Telosys Eclipse plugin (also known as "Telosys Studio")
is the most comfortable tool to setup a model, to create templates and to generate your code.
It provides specific editors for model entities, templates files (velocity files) and databases configuration.
It can be used to generate the code anywhere, so you can use it to generate your code in another IDE workspace.
Thanks to Eclipse Marketplace the installation is very easy.
This plugin is totally free and open source.

  • Entity file editor
  • Template file editor
  • Code generation
