Telosys Eclipse Plugin


Telosys Eclipse plugin is a comfortable tool for setting-up a model, creating templates and generating your code.
It provides editors for model entities and templates files.
It can be used to generate the code anywhere, so you can use it to generate your code in another IDE workspace.

Thanks to Eclipse Marketplace the installation is very easy.
This plugin is totally free and open source.

NB: Eclipse plugin currently only supports Telosys version 3.3.0
( for Telosys 4 we recommend VSCode + Telosys-CLI )
