Telosys CLI


Telosys-CLI is a Command-Line Interface that lets you run all Telosys operations using a set of simple commands.

You can use it to initialize a Telosys project, create a model, install or create bundles of templates and generate your code.

Telosys-CLI has been developed in Java, so the JRE (Java Runtime Environment) is the only requirement for using it.
This tool is totally free and open source.
For convenience, a VSCode extension is available. It is recommended for editing project files.
