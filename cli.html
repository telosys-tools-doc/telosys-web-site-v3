Telosys-CLI

Telosys CLI allows you to use all the Telosys core features with a simple and light Command Line Interface.
You can manage your models and generate your code by using only this CLI tool.
For ease of use there are extensions for Atom and VSCode editors.
Developed with Java, the JVM is the only requirement to use it.
This tool is totally free and open source.

  • Visual Studio Code Integration
  • Command Line Interface
  • Atom Integration
